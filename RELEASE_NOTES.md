## üîÑ Upgrade Guide

### Will My Current Theme Break?
**No!** All existing One Dark themes (regular, italic, vivid, vivid-italic) work exactly as before. This is a **100% backward-compatible** release.

### How Do I Try the New Themes?
1. Update the plugin from the JetBrains Marketplace
2. Go to `Preferences | Appearance & Behavior | Appearance`
3. Open the theme dropdown - you'll see 60 new themes!
4. Select one (e.g., "Dracula") and click OK
5. Don't like it? Switch back instantly - no commitment required!

### Do I Need to Configure Anything?
**Nope!** Just install the update and start exploring. Your current settings are preserved.

## üöÄ What You Can Do Now

### 1. Match Your Terminal
If you use Windows Terminal with a specific color scheme, you can now use the **exact same colors** in your IDE for a unified experience.

### 2. Try Something New
With 60+ themes, there's something for everyone:
- **Night owls:** Try Dracula, Tokyo Night, or Monokai
- **Productivity seekers:** Try JetBrains Darcula or GitHub Dark
- **Retro fans:** Try Zenburn, IR Black, or Wombat
- **Light theme users:** Try Solarized Light or Gruvbox Light

### 3. Import Your Own Theme
Have a custom Windows Terminal color scheme? You can now import it!
1. Save your scheme as JSON
2. Add it to `windows-terminal-schemes/`
3. Rebuild the plugin
4. Your custom theme appears in the dropdown

See [MIGRATION_GUIDE.md](docs/MIGRATION_GUIDE.md) for detailed instructions.
