## Color Scheme Format

All color schemes follow the Windows Terminal color scheme specification:

### Required Properties
- `name` - The display name of the color scheme
- `background` - Background color (hex format)
- `foreground` - Foreground/text color (hex format)
- `black`, `red`, `green`, `yellow`, `blue`, `purple`, `cyan`, `white` - ANSI colors 0-7
- `brightBlack`, `brightRed`, `brightGreen`, `brightYellow`, `brightBlue`, `brightPurple`, `brightCyan`, `brightWhite` - ANSI colors 8-15

### Optional Properties
- `cursorColor` - Color of the cursor
- `selectionBackground` - Background color for selected text

## Usage

These color schemes can be used for:
1. Testing the Windows Terminal to IntelliJ theme converter
2. Reference for color mapping algorithms
3. Example themes for documentation
4. Testing edge cases (light vs dark, high vs low contrast)
