# Windows Terminal Color Schemes Catalog

This directory contains **60 Windows Terminal color schemes** that can be imported and used to generate JetBrains themes.

## Overview

All schemes follow the Windows Terminal color scheme format with:
- 16 ANSI colors (black, red, green, yellow, blue, purple, cyan, white + bright variants)
- Background and foreground colors
- Optional cursor and selection colors

## Usage

### Importing a Scheme

1. Choose a scheme from the list above
2. The JSON file is located at: `windows-terminal-schemes/<scheme-name>.json`
3. Use the build system to generate a JetBrains theme from the scheme
4. The generated theme will be placed in the `themes/` directory

### Generating JetBrains Themes

```bash
# Generate all themes
./gradlew generateThemes

# The SchemaValidator will automatically validate each scheme during generation
```

### Validation

All schemes are validated using our `SchemaValidator` which checks:
- ✓ Hex color format validation
- ✓ WCAG contrast ratio checks
- ✓ Brightness distribution analysis
- ✓ Edge case detection (monochrome, high/low contrast)

## Adding New Schemes

To add a new scheme:

1. Create a JSON file in this directory with the Windows Terminal format
2. Follow the naming convention: lowercase with hyphens (e.g., `my-theme.json`)
3. Include all required fields: name, 16 ANSI colors, background, foreground
4. Optional fields: cursorColor, selectionBackground
5. Run validation: `./gradlew validateSchemes`
6. Update this SCHEMES.md file with the new scheme information
